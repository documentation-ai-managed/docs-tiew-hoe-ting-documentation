## Overview

Tiew Hoe Ting Documentation provides a centralized platform where you create, organize, and manage all your project documentation. You access a clean interface to build MDX-powered pages, collaborate with teams, and publish updates seamlessly. This space serves as your default hub for technical docs, guides, and API references.

## Contributing Code Examples

When adding custom components, use this pattern for multi-language setups.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Import and use a custom hook
  import { useDocs } from '@tiewhoeting/docs';

  function MyPage() {
    const { pages } = useDocs();
    return <div>{pages.length} docs available</div>;
  }
  ```
  ```python
  # Flask example for serving docs
  from flask import Flask
  app = Flask(__name__)

  @app.route('/docs')

      return {'message': 'Tiew Hoe Ting docs served'}
  ```
